# CIFX
Python framework for creating ciphers.

To use, clone this repository (or just download `cifx.py`) and place it in your project folder, then import:
```python
from cifx import CIFX
# your code...
```

lgpl 2.1

**note: This framework isn't designed for creating cryptographic ciphers, but for building cipher-languages. For example, if you want to create your own code-word language to conceal information from the uninitiated, you can write a program that encrypts plain text into your code-word language and vice versa, translating it back into plain text.**
